Project Setup

1. Clone repository
   git clone https://github.com/maryobimod/project-management.git

2. Masuk ke direktori project
   cd lara-collab

3. Install dependencies Composer
   composer install

4. Install dependencies NPM
   npm install

5. Copy file environment
   cp .env.example .env

6. Buat database kosong untuk aplikasi

7. Konfigurasi database
   Edit file .env dan sesuaikan konfigurasi database
   (DB_HOST, DB_DATABASE, DB_USERNAME, DB_PASSWORD)

8. Generate application key
   php artisan key:generate

9. Migrasi dan seeding database
   php artisan migrate --seed

   Saat diminta untuk seed development data,
   pilih "yes" untuk testing atau development.

10. Jalankan asset development
    npm run dev
    php artisan serve
